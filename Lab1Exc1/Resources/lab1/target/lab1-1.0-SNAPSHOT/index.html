Project Description:
For out project, we will be creating a functional online clothing store. Users will be able to browse items, add items to cart, see what items are in stock, be able to find items by category, check out, and create accounts.
Browse items: Extensive product catalog with categories (men, women, kids) with a search function.
Cart: Users will be able to add, remove, and edit their shopping cart
Stock: users will be able to see if items are sold out or how many are left in stock in each size
Users will be able to filter catalog by various different factors.
users will be able to check out. be able to insert their payment details.
users will be able to create accounts using their email. Admin will be able to edit and browse all accounts.
